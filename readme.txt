Mobile Call to Action plugin is used to add a custom Call to action button in the footer of your website. It can take 2 actions, one is for phone call and another one is for email button.

== Installation ==
Install the plugin from WordPress repository and activate it.
After activation, you will have to visit \"Mobile CTA Icons\" under the Settings menu from Your Dashboard. The settings are pretty much easy and informative.

== Frequently Asked Questions ==
Does it support on PC?
-No. It\'s only visible on Phone/Tabs
